# 💳 Banking Management System (CLI-based)

A simple Python-based CLI (Command Line Interface) banking system that allows users to create accounts, perform deposits and withdrawals, and manage transactions securely using file handling.

---

## 🚀 Features

- 🔐 Account Creation with unique 6-digit account number
- 🧾 User Authentication with password validation
- 💸 Deposit and Withdrawal operations
- 🧮 Real-time balance updates
- 🗂️ Transaction history logged using file handling
- 📁 Data stored persistently in local `.txt` files

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Interface:** Command Line Interface (CLI)  
- **Concepts Used:**  
  - File Handling  
  - Functions and Control Structures  
  - Date & Time Module  
  - Basic Authentication Logic

## ⚙️ How It Works

1. **Create Account**
   - Enter name, password, and initial deposit
   - System generates a unique 6-digit account number
   - Details saved in `accounts.txt`

2. **Login and Perform Transactions**
   - User authenticates with account number and password
   - Options to Deposit or Withdraw
   - New balance is updated in file and transaction logged

3. **Exit**
   - Safe exit from the system
